 autocomplete - дозволяє браузеру передбачати закінчення/продовження 
значення яке вводиь юзер в пошуковій строці

 autofocus -  надає перевагу якомусь елементу при загрузці він перший підгружається з сторінки, ясщо цей атрибут надати кнопці то її можна буде 
нажати навіть до повного завантаження сторінки клавішею ентер

 multiple -  дозволяє юзеру вводити більше ніз одне значення  в елемент, наприклад можливість вводити  два емейти чи завантажити  кілька файлів

placeholder - піказує юзеру що має бути введене в полі, діє поки юзер не почне вводити дані

 pattern (regexp) - забороняє вводити юзером в поле значення будьякі крім  вказаних атрибутом , наприклад коди країн

 readonly - робить поле вводу  лише  для читання без мождливості введення даних в це поле

 disabled - робить поле неактивним. сірого кольру  куди неможливоо внести дані чи інші зміни

 range -  призначений для створення шкали slidebar

 tel - створює поле призначене для номеру телефону юзера

 time - строковий елемент який представляє час, без інфи про часову зону 

 url - створює поле для введення url адреси, деякі браузери додають автоматично закінчення .com
 number - створює поле з можливість вводу чисел, справа в полі будуть кнопки зростання і спадання, ними можна задавати значення в полі , також можна настроїти  крок одного кліка тобто один клін не одиниця а будь яке число.

 dl - перший тег для створення списку з термінами і їх описами (<dt>, <dd>)
 dt - другий , середній тег який оприділяє термін що буде описуватись нижще
 dd - третій тег з описом терміну <dt> з списку <dl>

 fieldset - тег що створює велику форму для різних еоементів, коли потрібно вводити різні знічення в різну рількість полів, тег ці поля огортає (малює коробочку навколо них для зручності)
 legend - тег що оприділяє назву вищезгаданої форми, те що буде в формі (заголовок коробочки)

 datalist - тег що створює поле в якому є можливіть вибору  варіантів 

 optgroup - тег що створює поле в якому є можливіть вибору  варіантів, які є розділені по групам

 progress -тег що погазує шкалу прогресу, поточне і макс значення.

 meter- тег що погазує шкалу, наприклад задане значення 0.5 покаже напівзаповнену шкалу
 
iframe -  тег посилання який відкриває вікно на сторінці всередині якого відображається інформація з заданого в тегу посилання

 embed - тег використовується для відображення зовнішніх аплікацій , зображень , флешок чи відео, за допомогою посилання на них.
